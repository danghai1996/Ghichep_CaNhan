# Một số lệnh kiểm tra Switch

### Kiểm tra port status:
```
Switch#show interface status
```

### Show running config:
```
Switch#show running-config
```

### Kiểm tra VLAN
```
Switch#show vlan
```

### Kiểm tra VTP
```
SW# show vtp status

SW# show vtp pasword

SW# show vlan

SW# show interfaces trunk
```

### Kiểm tra mode port
```
switch# show interface switchport
```

hoặc
```
Switch#show interfaces fastEthernet 0/1 switchport
```

### Kiểm tra cấu hình Etherchannel
```
Switch#show etherchannel summary
```

### Kiểm tra cấu hình SW stack
Hiển thị danh sách SW
```
Stack# show switch
```
